# Certificate QR Handler

## Overview

This project is a backend application designed to handle certificate generation and QR code creation. It provides an API for managing certificates and generating QR codes for easy access and verification.

## Features

- Generate certificates
- Create QR codes for certificates
- Connect to a database for certificate storage

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Kalebb12/certificate-QR-handler.git
   ```
2. Navigate to the project directory:
   ```bash
   cd certificate-QR-handler
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```

## Usage

To start the application, run:

```bash
npm run dev
```
